# SPACE ATTACK
Destroy the buildings! But look out for the turrets. THEY ARE DEADLY. <br>
<br>
<br>
This game requires Unreal Engine 5.6 or above <br>
Not: No build is currently available

# Context
This project is for a university module in wair we had to make a game with a framework. <br>
It also has another game mode where you are a tank that destroys targets, but this was to learn the way around the engine.<br>
<br>
All assets were provided by the University of Staffordshire
